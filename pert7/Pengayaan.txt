Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Apa perbedaan antara interface dengan class biasa?
2. Mengapa interface tidak memiliki implementasi method di dalamnya?
3. Apa tujuan penggunaan keyword "implements" pada sebuah class?
4. Apakah sebuah class dapat mengimplementasikan lebih dari satu interface? Jelaskan dan berikan contoh sederhana.
5. Jika sebuah class mengimplementasikan interface, apa yang terjadi jika class tersebut tidak mengimplementasikan semua method dari interface tersebut?
jawaban

1. -Interface hanya berisi deklarasi method (tanpa implementasi), sedangkan class biasa bisa berisi implementasi lengkap
-Interface tidak dapat memiliki atribut non-static (kecuali static final), sedangkan class biasa bisa
-Class dapat mengimplementasikan banyak interface, tetapi hanya bisa mewarisi dari satu class parent
-Interface tidak dapat diinstansiasi langsung, sementara class biasa (non-abstract) bisa
-Interface mendefinisikan kontrak yang harus dipenuhi, sedangkan class biasa mendefinisikan implementasi

2. Interface dirancang untuk mendefinisikan kontrak atau spesifikasi perilaku yang harus diimplementasikan oleh class yang menggunakannya. 

3. Keyword "implements" digunakan untuk menunjukkan bahwa sebuah class berjanji untuk mengimplementasikan semua method yang dideklarasikan dalam interface yang diimplementasikan.

4. Ya,
public interface Printer {
    void print();
}
public interface Scanner {
    void scan();
}
public class MultiFungsiDevice implements Printer, Scanner {
    @Override
    public void print() {
        System.out.println("Printing document");
    }
    
    @Override
    public void scan() {
        System.out.println("Scanning document");
}
}

5. Program tidak akan dapat dikompilasi. Compiler Java akan menghasilkan error karena class 
tersebut belum memenuhi kontrak yang didefinisikan oleh interface.
